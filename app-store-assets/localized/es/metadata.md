## Description
`Arañazos de Gato te ayuda a mover contenido web de Safari a tu flujo de escritura.

Funciones:
• Capturar texto seleccionado o páginas completas
• Enviar contenido directamente a Drafts
• Usar la hoja de compartir cuando Drafts no esté disponible
• Personalizar plantillas, etiquetas, selectores CSS y filtros
• Sincronizar ajustes de la extensión a través de iCloud

Cómo funciona:
1. Activa la extensión Arañazos de Gato en Safari
2. Abre cualquier página web
3. Ejecuta Arañazos de Gato desde la barra de herramientas o un atajo de teclado
4. Continúa escribiendo en Drafts u otra app a través de la hoja de compartir

Arañazos de Gato está diseñado para escritores, investigadores y cualquiera que quiera un flujo de recorte web rápido y limpio.

Esta extensión solo accede a la página web actual cuando la activas. No rastrea el historial de navegación ni recopila datos.`
