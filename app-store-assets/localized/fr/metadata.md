## Description
`Griffures de Chat transfère le contenu web de Safari vers votre flux de rédaction.

Fonctionnalités :
• Capturer le texte sélectionné ou des pages entières
• Envoyer le contenu directement à Drafts
• Utiliser la feuille de partage quand Drafts n'est pas disponible
• Personnaliser les modèles, tags, sélecteurs CSS et filtres
• Synchroniser les réglages via iCloud

Comment ça marche :
1. Activez l'extension Griffures de Chat dans Safari
2. Ouvrez n'importe quelle page web
3. Lancez Griffures de Chat depuis la barre d'outils ou un raccourci clavier
4. Continuez à écrire dans Drafts ou une autre app via la feuille de partage

Griffures de Chat est conçu pour les rédacteurs, chercheurs et tous ceux qui veulent un flux de capture web rapide et propre.

Cette extension n'accède à la page web que lorsque vous la déclenchez. Aucun historique de navigation n'est suivi, aucune donnée n'est collectée.`
