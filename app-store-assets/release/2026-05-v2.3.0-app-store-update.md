# Cat Scratches 2.3.0 App Store Update

## What's New

Cat Scratches now does a much better job of finding the real story text and leaving page chrome behind.

- Improved article detection across a broader capture test corpus
- Better filtering for ads, video controls, newsletter prompts, related links, sidebar blocks, and other non-story clutter
- Cleaner Drafts output with fewer repeated sections and less navigation noise
- Stronger text cleanup for boilerplate that remains after page filtering
- Added regression checks to keep capture quality stable across future updates

## Review Notes

No account or sign-in is required.

How to test:
1. Install Cat Scratches and enable the Safari extension.
2. Open a story or article page in Safari.
3. Run Cat Scratches from Safari.
4. Confirm the resulting Draft contains the article title, source URL, and clean body text with minimal page chrome.

Expected behavior:
- If Drafts is installed, captured content opens in Drafts.
- If Drafts is unavailable, the Share Sheet fallback remains available.
